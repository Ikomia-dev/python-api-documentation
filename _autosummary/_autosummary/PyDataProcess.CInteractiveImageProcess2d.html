PyDataProcess.CInteractiveImageProcess2d¶

class PyDataProcess.CInteractiveImageProcess2d¶

Add user interactions capability to a 2D image process task. The class implements a user interaction mechanism through the use of dedicated graphics layer. When a CInteractiveImageProcess2d instance is active, the system automatically activates this internal graphics layer on which the user can interact by drawing items (points, lines, polygons…). Every changes on this layer are then notified to this class, and actions can be implemented accordingly. The class is used for example to handle interactive segmentation with color picker.

Derived from CImageProcess2d.
