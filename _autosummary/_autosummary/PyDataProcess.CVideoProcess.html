PyDataProcess.CVideoProcess¶

class PyDataProcess.CVideoProcess¶

Add video specific features. This class defines a task with the following properties:

Inputs:

  • video (PyCore.CVideoProcessIO)

  • graphics layer (not accessible yet in Python API)

Outputs:

  • video (PyCore.CVideoProcessIO)

It should be the base class for all task dedicated to video processing.

Derived from CImageProcess2d.
