CIkomiaRegistry¶

class ikomia.dataprocess.pydataprocess.CIkomiaRegistry¶

Algorithms registry of the Ikomia platform.

Class that enables to instanciate every algorithm from its name. CIkomiaRegistry objects communicate automatically with the Ikomia Marketplace to download algorithm package if necessary. It also include dedicated function to register new algorithms

Note

A pure Python derived class is also implemented to add higher level features. See IkomiaRegistry for more information.

Import

from ikomia.dataprocess import CIkomiaRegistry
