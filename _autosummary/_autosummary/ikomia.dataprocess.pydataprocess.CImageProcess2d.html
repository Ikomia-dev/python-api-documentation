CImageProcess2d¶

class ikomia.dataprocess.pydataprocess.CImageProcess2d¶

Implement all basic features used in a task dedicated to 2D image processing. This class defines a task with the following properties:

Inputs:

  • image (CImageProcessIO)

  • graphics layer (not accessible yet in Python API)

Outputs:

  • image (CImageProcessIO)

It is a good starting point to use it as a base class for all task dedicated to 2D images.

Derived from CProtocolTask.

  • height (int) – mask height (should be the height of the source image)

  • graphics (CGraphicsInput) – graphics items become white blobs in the black background mask

setOutputColorMap((CImageProcess2d)self, (int)index, (int)mask_index, (object)colors) → None :¶

Bind a display color map to an image output. The color mask is generated from an output mask generated by the task itself (binary or labelled image).

Parameters

  • index (int) – zero-based index of the output to be displayed with the color map. The output must be a CImageProcessIO or derived.

  • mask_index (int) – zero-based index of the output representing the mask used to generate the color overlay.

  • colors (list) – list of tuples (r,g,b values) for the color map. If empty, the system generates random colors.
