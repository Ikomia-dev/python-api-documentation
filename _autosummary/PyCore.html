PyCoreÂ¶

Module offering API core objects such as task, I/O, parameters and widget.

The extension capability of the Ikomia platform is based on inheritance and this module offers the main base classes. The system uses the concept of protocol (ie worflow) to process images. A protocol is a graph composed of processing tasks, each task comes with its parameters, its inputs and its outputs. Additionnally, a widget (for Graphical User Interface) is associated with the task to ensure user interaction. Through this API we provide base class that you could override for each component.
