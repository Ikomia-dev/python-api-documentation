PyCore.CGraphicsInput¶

class PyCore.CGraphicsInput¶

Define task input containing graphics items. Consult GraphicsItem to see the list of possible graphics types. Instance can be added as input of a CProtocolTask or derived. This kind of input is used to manage user-defined ROI or to forward graphics items generated by a previous task in your workflow. Derived from CProtocolTaskIO.
