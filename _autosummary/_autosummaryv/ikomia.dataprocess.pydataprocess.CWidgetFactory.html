CWidgetFactory¶

class ikomia.dataprocess.pydataprocess.CWidgetFactory¶

Abstract class defining the core structure of the process widget factory. The system extensibility for the process library is based on the factory design pattern. Each process task must implement a widget factory class derived from this class. Then the system is able to instantiate dynamically a process widget object (even user-defined).
