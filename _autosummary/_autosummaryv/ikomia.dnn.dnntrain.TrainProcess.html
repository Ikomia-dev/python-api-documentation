TrainProcess#

class ikomia.dnn.dnntrain.TrainProcess(name, param)#

Base class for task dedicated to Deep Learning training. It includes MLflow framework and Tensorboard and handle connections with them:

  • experiment creation

  • runs management

  • parameters logging

  • metrics logging

  • training dashboard

It must be use with TaskParam or derived for parameters. Derived from CDnnTrainTask.
