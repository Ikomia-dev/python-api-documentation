TrainProcess¶

class ikomia.dnn.dnntrain.TrainProcess(name, param)¶

Base class for task dedicated to Deep Learning training. It includes MLflow framework and handle connections with it:

  • experiment creation

  • runs management

  • parameters logging

  • metrics logging

  • training dashboard

It must be use with TrainParam or derived for parameters.
