CWorkflowTaskWidget

class ikomia.core.pycore.CWorkflowTaskWidget

Base class for all widgets associated with a task. Each process comes with a Qt widget to manage algorithm parameters and user actions. The widget class implementation is mandatory for Ikomia Studio. This class offers some basic tools to build rapidly a functionnal user interface. As it is derived from Qt QWidget, one can easily customize it to fit more complex needs. Use of bindings like PyQt5 or PySide2 are recommended.

Import

from ikomia.core import CWorkflowTaskWidget
