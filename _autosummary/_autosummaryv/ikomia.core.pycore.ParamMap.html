ParamMap¶

class ikomia.core.pycore.ParamMap¶

Data structure (same as Python dict) to store task parameters names and values

Example:

from ikomia import core

paramMap = core.ParamMap()
paramMap["parameter1"] = str(self.parameter1)
paramMap["parameter2"] = str(self.parameter2)
paramMap["parameter3"] = str(self.parameter3)
