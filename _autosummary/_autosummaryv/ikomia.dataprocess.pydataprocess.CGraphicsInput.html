CGraphicsInput

class ikomia.dataprocess.pydataprocess.CGraphicsInput

Define task input containing graphics items. Consult GraphicsItem to see the list of possible graphics types. Instance can be added as input of a CWorkflowTask or derived. This kind of input is used to manage user-defined ROI or to forward graphics items generated by a previous task in your workflow. Derived from CWorkflowTaskIO.

getUnitElementCount((CWorkflowTaskIO)self) → int :

Get the number of unit element in terms of processing scheme. This value is used to define the number of progress steps for progress bar component in Ikomia Studio. For an image, the count is 1. For Z-stack volume, the count is the number of Z levels. Should be overriden for custom input or output.

Returns:
