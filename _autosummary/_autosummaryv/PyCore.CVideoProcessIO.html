PyCore.CVideoProcessIO¶

class PyCore.CVideoProcessIO¶

Define an input or output for a task dedicated to video management. This class is designed to handle video and instance can be added as input or output to a CProtocolTask or derived object. It is the base class to define input or output of a video processing task. Several video output type can be defined according to the nature of the algorithm:

  • binary video

  • labelled video (graylevel connected components)

  • standard video

Source video can be either a file, an image sequence or a stream. Image data is stored as a numpy array.

setOverlayMask((CImageProcessIO)self, (object)mask) → None :¶

Set the associated overlay mask. Ikomia software is able to display overlay mask on top of image. This method sets this mask, it will be displayed automatically according to a predefined color map. Zero pixels of the mask will be completely transparent, non-zero will be displayed according to the corresponding color in the color map. The color map must be defined in the task implementation. See PyDataProcess.CImageProcess2d.setOutputColorMap() for details.

Parameters

image (Numpy array) – image buffer (8 bits - 1 channel)
