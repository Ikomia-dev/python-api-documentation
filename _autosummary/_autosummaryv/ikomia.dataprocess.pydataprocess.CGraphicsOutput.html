CGraphicsOutput¶

class ikomia.dataprocess.pydataprocess.CGraphicsOutput¶

Define task output containing graphics items. Consult GraphicsItem to see the list of possible graphics types. Instance can be added as output of a CProtocolTask or derived. This kind of output is used to manage graphics items generated in a task. Ikomia software displays it as an overlay layer on top of images or videos. Graphics items can then be forwarded as input of workflow’s following tasks. Derived from CProtocolTaskIO.

getUnitElementCount((CProtocolTaskIO)self) → int :¶

Get the number of unit element in terms of processing scheme. This value is used to define the number of progress steps for progress bar component. For an image, the count is 1. For Z-stack volume, the count is the number of Z levels. Should be overriden for custom input or output.

Returns:

int: number of unit element to process.
