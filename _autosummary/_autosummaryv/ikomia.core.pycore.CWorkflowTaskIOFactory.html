CWorkflowTaskIOFactory¶

class ikomia.core.pycore.CWorkflowTaskIOFactory¶

Abstract class defining the core structure of the task I/O factory. The system extensibility is based on the factory design pattern. Each task input/output must implement a factory class derived from this class. Then the system is able to instantiate dynamically a I/O object (even user-defined).

Import

from ikomia.core import CWorkflowTaskIOFactory
