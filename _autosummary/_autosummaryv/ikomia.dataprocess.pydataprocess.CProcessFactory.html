CProcessFactory¶

class ikomia.dataprocess.pydataprocess.CProcessFactory¶

Abstract class defining the core structure of the process factory. The system extensibility for the process library is based on the factory design pattern. Each process task must implement a factory class derived from this class. Then the system is able to instantiate dynamically a process object (even user-defined).
