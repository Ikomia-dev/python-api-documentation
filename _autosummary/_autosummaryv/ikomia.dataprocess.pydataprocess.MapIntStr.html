MapIntStr

class ikomia.dataprocess.pydataprocess.MapIntStr

Data structure (same as Python dict) to store int key and string value

Example:

from ikomia.dataprocess import MapIntStr

categories = MapIntStr()
categories[0] = "boat"
categories[1] = "kyte"
categories[2] = "windsurf"
categories[3] = "surf"
