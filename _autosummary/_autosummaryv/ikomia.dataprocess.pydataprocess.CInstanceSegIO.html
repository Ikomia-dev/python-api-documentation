CInstanceSegIO

class ikomia.dataprocess.pydataprocess.CInstanceSegIO

Define input or output managing common information extracted by instance segmentation task. Such task are able to automatically detect objects and their shapes in image. It gets class label, confidence, bounding box and mask for each one. For each object, information is stored in a CInstanceSegmentation instance. Among others, algorithms like MaskRCNN, Yolact, SparseInst are instance segmentation tasks.
