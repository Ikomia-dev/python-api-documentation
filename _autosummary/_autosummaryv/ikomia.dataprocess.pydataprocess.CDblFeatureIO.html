CDblFeatureIO¶

class ikomia.dataprocess.pydataprocess.CDblFeatureIO¶

Define input or output for a task dedicated to manage numeric values. C++ implementation uses template to handle various data types. In Python, Ikomia API exposes only the double precision (float) specialization. The class is designed to handle numeric values, it consists on a list of float values, a list of associated labels and a display type (see NumericOutputType). For the specific case of plot display, a plot type property is available (see PlotType). Derived from CProtocolTaskIO
