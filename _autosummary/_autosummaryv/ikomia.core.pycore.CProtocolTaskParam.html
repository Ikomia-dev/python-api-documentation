CProtocolTaskParam¶

class ikomia.core.pycore.CProtocolTaskParam¶

Base class for the management of task parameters. It consists of two virtual methods that should be overriden to fill or get the structure holding parameters value. This structure is used internally by the system to save and load parameters values of task used in a workflow.
